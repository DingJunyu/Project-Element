# Project-Element
アクションRPG調合システム検証ゲームーープロジェクト元素  

今までたくさんのゲームをやった上で、遺憾と思っているポイントはたくさんありますが、今回の企画で検証したい課題は：  
「集めたアイテムはすべて使える、自分で自分なりな攻撃を定義しましょう！」となります。  

ＲＰＧゲームをプレイする時に、ゴミをいっぱい集まったケースも少なくないと思いますし、集まったものはレシピ通り特定なものしか調合できません。  
今の企画は戦闘アイテムから、プレーヤーが使いやすい調合システムを完成したいです。  
その上に戦闘の部分と二つぐらいのステージを完成しましょう！
